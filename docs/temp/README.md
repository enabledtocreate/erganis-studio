# Temporary documentation (Studio)

Staging area for Studio and Client app notes, module design drafts, and document fragments **before** APM-managed docs exist for this repo.

## Use this folder for

- Studio / Client UX and RBAC notes
- First-party module docs (`modules/documents/`, `modules/inventory/`, etc.)
- Shared UI (`shared/`) and integration spikes
- Content destined for APM fragments

## Do not use for

- Hand-editing final managed docs in-repo (use APM when available)
- Secrets or `.env` values

## Cleanup

When a topic is promoted to APM or the product plan, remove or archive the scratch file here.
